Tags And Element And Attribute

Tags

Which comes in between <Angular Brackets>

Tags are of two types :-

1. Which comes in pair (Starting Tag and Closing Tag)

eg. => h1, p, title etc

2. Self Closing tags / Void tag / Empty Tags

eg. => hr, br input etc

Element

Element =>Start Tag - Some Content - End Tag

Attribute

Attribute => It provide Additional information, after that behaviour of Element will change

Always Written in Start Tag
